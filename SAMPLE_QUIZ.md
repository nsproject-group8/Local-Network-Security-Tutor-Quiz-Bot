# Network Security Sample Quiz Questions

## Sample questions the bot can generate after ingesting documents

### Multiple Choice Questions (MCQ)

1. What is the primary purpose of a firewall?
   A) To encrypt network traffic
   B) To monitor and control incoming and outgoing network traffic
   C) To detect viruses
   D) To backup data
   **Answer: B**

2. Which encryption type uses the same key for encryption and decryption?
   A) Asymmetric encryption
   B) Public key encryption
   C) Symmetric encryption
   D) Hash encryption
   **Answer: C**

3. What does VPN stand for?
   A) Virtual Private Network
   B) Very Protected Network
   C) Variable Protocol Network
   D) Verified Private Node
   **Answer: A**

### True/False Questions

1. A DDoS attack attempts to make a network resource unavailable by overwhelming it with traffic.
   **Answer: True**

2. Symmetric encryption uses different keys for encryption and decryption.
   **Answer: False**

3. Multi-factor authentication is more secure than single-factor authentication.
   **Answer: True**

4. An Intrusion Detection System (IDS) can only detect attacks, not prevent them.
   **Answer: True**

### Open-Ended Questions

1. Explain the difference between symmetric and asymmetric encryption, providing examples of each.

2. Describe how a Man-in-the-Middle (MITM) attack works and what measures can be taken to prevent it.

3. What are the main components of defense-in-depth strategy in network security?

4. Explain the role of certificates in SSL/TLS communication.
